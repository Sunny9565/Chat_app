Disposable Temporary Chat Application
=========================================

Features Included:
1. Zero Authentication: No login or signup required.
2. Direct Connection: Generate a clean 6-digit Unique Room ID and share it directly with a friend.
3. 20-Hour Expiration: Active channels are automatically hard-deleted after 20 hours.
4. Clean Real-time Interface: Fully optimized layout with responsive dark mode styling.

Instructions to Run Locally:
----------------------------
1. Make sure you have Node.js installed on your computer.
2. Extract this ZIP file into a folder.
3. Open your terminal or command prompt inside that folder.
4. Install dependencies by running:
   npm install
5. Start the chat server by running:
   npm start
6. Open your web browser and go to:
   http://localhost:3000
